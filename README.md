# Подготовка данных
Мне были предоставлены изображения лиц людей размером 256×256 пикселей:

- **Реальные лица** - фотографии людей с сайта Flickr

- **Синтетические лица** - сгенерированы с помощью StyleGAN

# Архитектура модели
В основе решения лежит архитектура Xception. Основная особенность данной архитектуры — использование глубинно-разделяемых сверток (Depthwise Separable Convolutions), что позволяет эффективно извлекать пространственные признаки при меньшем количестве параметров по сравнению с классическими свертками. В финальный слой добавлен Dropout (0.5) для предотвращения переобучения.

# Параметры обучения
- **Оптимизатор**: Adam с начальным шагом обучения $4 \times 10^{-4}$.
- **Функция потерь**: `BCEWithLogitsLoss` (с использованием pos_weight для компенсации дисбаланса классов).
- **Планировщик**: `ReduceLROnPlateau` (уменьшение LR в 2 раза при стагнации лосса на валидации в течение 2 эпох).
- **Регуляризация**: Gradient Clipping (1.0) для стабилизации градиентов.
- **Оптимизация порога**: Вместо стандартного значения 0.5, в коде реализован цикл поиска оптимального порога на валидационной выборке. Это позволило поднять F1-score на несколько пунктов, адаптируя модель под специфику распределения предсказаний.
